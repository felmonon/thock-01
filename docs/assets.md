# Asset Provenance and Licences

| Asset | Method | Rights/provenance |
| --- | --- | --- |

| Force curves | Original inline SVG paths representing tactile, linear, and clicky profiles. | Created for this entry. |
| Soundprint traces | Deterministic SVG geometry generated from visitor cadence, available pointer pressure, and selected profile. | Created at runtime by project code. |
